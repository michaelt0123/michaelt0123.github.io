Keyboard Shortcuts

Function Keys:

F5 = Refreshes the Content of Open Browser and Explorer Windows that have Focus.

Key Combos:

Interacting with Text & File Explorer Content:

Ctrl + A = Select All
Ctrl + X = Cut
Ctrl + C = Copy
Ctrl + V = Paste
Ctrl + Z = Undo
Ctrl + Y = Redo
Ctrl + F = Find

Accessing Utilities:

Ctrl + Alt + Del = Change LMI Password / Access Task Manager
Ctrl + Shift + Esc = Launch Task Manager
Alt + Tab = Switch Between Apps (By tabbing through open apps)
Win + D = Show the Desktop (Again to Show Open Apps)
Win + E = Open Windows Explorer
Win + L = Lock the Computer
Win + R = Open the Run Utility
Win + X = Open the System Menu
Win + Shift + S = Open the Snipping Tool
Win + Pause/Break = Open the System Information Window

Windows Virtual Desktops:

Win + Ctrl + D = Open a Virtual Desktop
Win + Ctrl + <- or -> arrow = Cycles through Virtual Desktops
Win + Ctrl + F4 = Close a Virtual Desktop

Other Helpful Key Combinations:

Ctrl + Shift + T = Reopen browser tabs that have been closed.
