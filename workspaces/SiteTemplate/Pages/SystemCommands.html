Common Windows Commands

Retrieve System Information:

# Display the Hostname of the Local System # hostname
# Display Local System Information # systeminfo
# Display Remote System Information # systeminfo /s Computername
# Display File Extension Associations # assoc
# Display the Installed Drivers # driverquery -v
# Display Currently Running Tasks # tasklist
# End a Running Task by providing it's Image Name or the Process ID # taskkill
# List the Global Switches and Aliases available for the wmic Command # wmic /?
# List the Properties available for a selected wmic Alias # wmic bios get /?
# Get the Serial Number of the Local Computer # wmic bios get serialnumber
# Get the Serial Number of a Remote Computer # wmic /node:Computername bios get serialnumber
# Get the Description and Size of Installed Hard Drives # wmic logicaldisk get deviceid, volumename, description, size, drivetype, freespace
# Get the Manufacturer, Model, S/N, and Total Size of the HDD # wmic diskdrive get manufacturer, model, serialnumber, size
# Display the Volume Label and S/N of a HDD # vol C:
# Check with the Domain Controller for new or updated Group Policies and apply them. # gpupdate /force
# Display all Group Policy objects for the current user and the local system. # gpresult /r

Retrieve Network Information:

# Display all TCP/IP network configuration values # ipconfig /all
# Force the local system to give up it's IP address. # ipconfig /release
# Instruct the local system to request a new IP address. # ipconfig /renew
# Clear the DNS cache of the local system. # ipconfig /flushdns
# View the network connections that are active between your system and any other systems on your network or the internet and display the listening ports. # netstat -a
# Display the names that have been registered locally on the system by NetBIOS applications. # nbtstat -n
# Use the Ping command to test the network communication between your pc and another IP address or URL. # ping 127.0.0.1
# View the network communication path between your pc and another IP address or URL. # tracert 127.0.0.1
# Query the DNS to obtain IP address mapping or domain name information. (Note: This can be used to see the IP address for any URL.) # nslookup 127.0.0.1
# Display the MAC addresses of the network interfaces. # getmac
# Retrieve the Address Resolution cache. # arp -a
# Display the active state of TCP Global Parameters. # netsh interface tcp show global
# Display all of the WiFi profiles that have been connected to. # netsh wlan show profile
# View the WiFi password of a WiFi profile. (Note: Replace "ProfileName" with the name of the WiFi profile.) # netsh wlan show profile ProfileName key=clear

Perform Other Actions:

# View Help Information for Windows Commands # help
# Clear the Contents of the Console Window # cls
# View the History of Commands that have been executed in the Console Window. # doskey /history
# Clear the History of Commands that have been executed in the Console Window. (Note: The Console Window must have focus for this to work.) # Alt + F7
# Send Command Output to the Clipboard # Command | clip
# Send Output to a File # Command > FilePath\FileName.ext
# Append Output to a File # Command >> FilePath\FileName.ext
# Change the Prompt. # prompt ADesiredCombinationOfTextAndOr $Options
# Changing the Prompt back to Normal. # prompt $P$G
# View More Prompt Options. # prompt /?
# Display the Current working directory. # cd
# List files and folders in the Current working directory. # dir
# List all files in the Current working directory that are of a specific file type. # dir *.ext
# Change to a different working directory. (Example: cd C:\Windows or cd C:\[TAB-key] to view folders in within a directory.) # cd Volume:\FolderPath\FolderName
# Navigate to the immediate parent node of the Current working directory. # cd ..
# Create a new folder in the Current working directory. # mkdir FolderName
# Remove an empty folder from the Current working directory. (Note: The "rd" command performs the same action.) # rmdir FolderName
# Remove a folder and all of it's contents from the Current working directory. # rm -r FolderName
# Move a file from the Current working directory to a different directory. # move FileName.ext Volume:\FolderPath\FolderName
# Rename a file in the Current working directory. # ren FileName.ext DifferentFileName.ext
# Display the attributes of all files recursively in the Current working directory. # attrib /s
# Create a File in the Current directory. (Note: Press Enter while writing text to go to the next line.) # copy con Filename.ext
# Save the contents of the copy con command to the file that is being created. # Ctrl + Z
# Break out of a running process. # Ctrl + C
# Close the Console Window. # exit
# Logoff the Computer. # shutdown /l
# Restart the Computer. # shutdown /r
# Shutdown the Computer. # shutdown /s
